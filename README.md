🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript.
This project showcases my skills, projects, and provides a contact form for communication.

🚀 Features
✨ Responsive and modern UI design
👩‍💻 About section with skills progress bars
🛠️ Services / Skills section
📂 Projects portfolio showcase
📞 Contact form with Google Sheets integration
🔗 Social media links (GitHub & LinkedIn)
🎨 Smooth scrolling and interactive UI
🛠️ Tech Stack
HTML5 – Structure of the website
CSS3 – Styling and layout
JavaScript (Vanilla JS) – Form handling and interactivity
Font Awesome – Icons
Google Apps Script – Backend for contact form

⚙️ How It Works
Contact Form Integration

The form uses Google Apps Script as backend:

Sends form data using fetch()
Displays success/error message dynamically
Resets form after submission
fetch(scriptURL, {
  method: "POST",
  body: new FormData(form)
})
